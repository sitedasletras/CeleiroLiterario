function normalizarLinha(l){ return String(l || '') .trim() .normalize('NFD') .replace(/[\u0300-\u036f]/g,'') .toLowerCase(); } function limparSaida(t){ const original = String(t || '').replace(/\r/g,''); const linhas = original.split('\n'); let inicio = 0; let fim = linhas.length; const idxSumario = linhas.findIndex(l => /^\s*sum[aá]rio\s*:?$/i.test(l.trim()) ); if(idxSumario >= 0){ inicio = idxSumario + 1; while(inicio < linhas.length && !linhas[inicio].trim()){ inicio++; } } const marcadoresFim = [ 'biografia', 'sobre o autor', 'o autor', 'sobre a autora', 'a autora', 'dados do autor', 'dados da autora' ]; for(let i = inicio; i < linhas.length; i++){ const n = normalizarLinha(linhas[i]); if(marcadoresFim.includes(n)){ fim = i; break; } } let trecho = linhas .slice(inicio, fim) .join('\n'); trecho = trecho .replace(/\n{4,}/g, '\n\n\n') .trim(); return trecho || original; }
