# Núcleo de Elenco Heteronímico Mundial SIGMAL

Ferramenta institucional de uso exclusivo de Wagner Planas, vinculada aos perfis Admin, Heterônimo e Wagner Planas.

Este núcleo não pertence à OHE, ao Lapidar ou ao Caminho das Pedras. Ele é uma ferramenta própria do eixo SIGMAL / Web TV Celeiro / Web Rádio Celeiro.

## Função

Criar, organizar e preservar atores e atrizes heteronímicos virtuais permanentes para:

- Web TV Celeiro Literário
- Web Rádio Celeiro Literário
- Silos Cinematográficos SIGMAL
- trailers literários
- curtas institucionais
- vinhetas
- vídeos musicais
- programas narrativos
- materiais audiovisuais institucionais

## Regra de continuidade

Cada ator ou atriz deve manter:

- nome artístico permanente
- rosto reconhecível permanente
- biotipo corporal base permanente
- território estético definido
- estilo de atuação definido
- compatibilidade narrativa definida

Podem variar:

- figurino
- cabelo
- barba
- maquiagem
- iluminação
- cenário
- leve variação de peso

Não podem variar:

- identidade facial central
- assinatura dos olhos
- assinatura do nariz
- assinatura da boca
- identidade artística geral

## Perfis com acesso

- Admin
- Heterônimo
- Wagner Planas

Rovonilson Reigns Bautista: narrador, apresentador e artista musical; idade-base 54 anos; perfil visual derivado da imagem de Wagner Planas com liberdade controlada de tom de pele, cabelo, barba e presença cênica.
